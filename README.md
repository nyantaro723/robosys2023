# plusコマンド

![test](https://github.com/nyantaro723/robosys2023/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足す。

## 使い方

ディレクトリ: $ cd GIT/robosys2023/

実行例: $ seq 10 | ./plus


## 必要なソフトウェア

* Python
  * テスト済みバージョン: 3.7～3.10

## テスト環境

* Ubuntu 20.04
